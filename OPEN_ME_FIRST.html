Sudoku Factory / ナンプレ工場 Ver.0.2

スマートフォンでできるだけスクロールせずに遊べる、広告なしの静的ナンプレPWAです。

作成したもの

Ver.0.2では、盤面を大きくし、ヘッダー、問題選択、メッセージ、数字ボタン、操作ボタンをコンパクトにしました。選択中のマス、同じ行、同じ列、同じ3x3ブロック、同じ数字もハイライトされます。

ローカルで開く方法

同じフォルダの index.html をブラウザで開くと遊べます。PWAのオフライン機能まで確認する場合は、ローカルWebサーバーまたはGitHub Pagesから開いてください。

GitHub Pages URL

https://yuwakobo-ux.github.io/sudoku-factory/

遊び方

問題を選び、空いているマスをタップしてから、下の数字ボタンで1から9を入力します。最初から入っている数字は変更できません。

ミス確認のテスト

空きマスにわざと違う数字を入れて、ミス確認 を押してください。間違っている入力済みマスが赤く表示されます。空きマスはミス扱いになりません。

ヒントのテスト

ヒント1、ヒント2、ヒント3 を押すと、現在空いているマスから指定数だけ正しい数字が入ります。

既知の制限

Ver.0.2 は固定問題3つのみです。メモ機能、候補数字、タイマー、問題生成、アカウント、広告、バックエンドは入れていません。ブラウザのツールバーが大きい小型端末では、少しスクロールが残る場合があります。
