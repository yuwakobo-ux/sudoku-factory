Sudoku Factory / ナンプレ工場 Ver.0.4

Ver.0.4は機能追加ではなく、実際に遊ぶときの見やすさを大きく改善するUIポリッシュ版です。

Ver.0.4の改善点

選択中のマスは濃い緑、太い枠、影、少しの拡大で強く目立ちます。同じ数字は黄色、同じ行と列は青、同じ3x3ブロックは薄い緑、間違いは赤で表示します。

レビュー確認リスト

  • 選択中のマスを一瞬で見つけられますか？
  • 同じ数字を一瞬で見つけられますか？
  • 選択中の行と列を一瞬で見分けられますか？
  • 3x3ブロックが行・列と違う見え方になっていますか？
  • ミスの赤いマスはすぐ分かりますか？
  • Ver.0.3より盤面が読みやすくなっていますか？

数字ボタンの確認

数字が入っているマスを選ぶと、同じ数字のボタンが黄色く強調されます。盤面に9個入った数字は薄く表示されますが、操作性を守るため完全には無効化していません。

ローカルで開く方法

同じフォルダの index.html をブラウザで開くと遊べます。PWAのオフライン機能まで確認する場合は、ローカルWebサーバーまたはGitHub Pagesから開いてください。

GitHub Pages URL

https://yuwakobo-ux.github.io/sudoku-factory/

既知の制限

Ver.0.4 は固定問題3つのみです。メモ機能、候補数字、タイマー、問題生成、アカウント、広告、バックエンドは入れていません。ブラウザのツールバーが大きい小型端末では、少しスクロールが残る場合があります。

次のおすすめタスク

スマートフォン実機で盤面を数分プレイし、選択、同じ数字、行・列、ミスの見分けやすさを確認するのがおすすめです。
