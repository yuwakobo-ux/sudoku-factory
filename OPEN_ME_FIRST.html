Sudoku Factory / ナンプレ工場 Ver1.0

作成内容

Ver1.0 Complete Editionとして、Easy 20、Normal 20、Hard 20、合計60問に増やしました。新しいゲーム機能は追加せず、進捗管理と問題一覧を整えています。

確認URL

https://yuwakobo-ux.github.io/sudoku-factory/

キャッシュが残る場合は ?v=ver1.0 を付けて開いてください。

レビュー項目

  • Easy、Normal、Hard がそれぞれ 20 問ありますか？
  • Total が 60 問になっていますか？
  • Home に Easy / Normal / Hard / Total / Completion が表示されていますか？
  • 問題一覧に 001 から 020 と状態が表示されていますか？
  • 途中入力後、状態が playing になりますか？
  • 完成確認で cleared になりますか？
  • 続きボタンで最後の playing 問題に戻れますか？
  • 現在の問題だけリセットできますか？
  • 全進捗リセットは確認ダイアログ付きですか？

検証コマンド

node --check app.js
node --check puzzles.js
node --check service-worker.js
node verify-static.js

既知の制限

メモ、タイマー、問題生成、広告、バックエンド、アカウントはありません。問題は固定の静的データです。
