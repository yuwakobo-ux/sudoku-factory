Sudoku Factory / ナンプレ工場 Ver.0.5

Ver.0.5は高コントラストの見た目 redesign です。機能追加ではなく、実機で変化が分かることを優先しました。

Ver.0.5の変更点

選択中のマスは強い青、同じ数字は黄色、行と列は水色、3x3ブロックは薄い緑、間違いは赤で表示します。色の役割をVer.0.4より大きく離しました。

レビュー確認リスト

  1. 選択中のマスははっきり目立ちますか？
  2. 同じ数字は明確に黄色で見えますか？
  3. 行・列と3x3ブロックは違う色に見えますか？
  4. Ver.0.4と比べて明らかに違う見た目ですか？

スマートフォンで確認すること

数字が入っているマスを選び、選択セル、同じ数字、行、列、ブロックが一瞬で分かるか確認してください。ミス確認では、赤いセルがすぐ見つかるか確認してください。

ローカルで開く方法

同じフォルダの index.html をブラウザで開くと遊べます。PWAのオフライン機能まで確認する場合は、ローカルWebサーバーまたはGitHub Pagesから開いてください。

GitHub Pages URL

https://yuwakobo-ux.github.io/sudoku-factory/

既知の制限

Ver.0.5 は固定問題3つのみです。メモ機能、候補数字、タイマー、問題生成、アカウント、広告、バックエンドは入れていません。小さい画面では凡例を非表示にします。
