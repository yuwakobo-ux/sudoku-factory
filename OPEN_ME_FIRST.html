変更点

  • Hint 2 と Hint 3 を削除しました。
  • ヒントは1つだけにしました。
  • Difficulty と Puzzle の2つの選択欄に分けました。
  • 選択中の問題の状態を new / playing / cleared で表示します。
  • 進捗表示を Easy / Normal / Hard / Total に整理しました。
  • 操作ボタンを2段に整理しました。

レビュー項目

  • 難易度を Easy / Normal / Hard で選べますか？
  • 問題番号を 001 から 020 で選べますか？
  • 選択中の問題状態が分かりますか？
  • 続きからボタンの意味が分かりやすいですか？
  • Hint 2 / Hint 3 が消えていますか？
  • 盤面が大きいまま保たれていますか？

検証コマンド

node --check app.js
node --check puzzles.js
node --check service-worker.js
node verify-static.js
