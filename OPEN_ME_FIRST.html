Sudoku Factory / ナンプレ工場 Ver.0.3

Ver.0.3では、スマートフォンの1画面レイアウトを保ったまま、色の見分けやすさを改善しました。

Ver.0.3の色改善

選択中のマスは濃い緑、同じ数字は淡い黄色、同じ行と列は淡い青、同じ3x3ブロックは淡い緑系、間違いは赤系で表示します。固定数字は濃く太い文字、入力した数字は青緑の文字です。

スマートフォンで確認すること

マスを選んだときに、選択マス、同じ数字、行、列、ブロックの違いがすぐ分かるか確認してください。数字ボタンでは、選択中の数字が黄色になり、盤面に9個入った数字が薄く表示されます。

ローカルで開く方法

同じフォルダの index.html をブラウザで開くと遊べます。PWAのオフライン機能まで確認する場合は、ローカルWebサーバーまたはGitHub Pagesから開いてください。

GitHub Pages URL

https://yuwakobo-ux.github.io/sudoku-factory/

遊び方

問題を選び、空いているマスをタップしてから、下の数字ボタンで1から9を入力します。最初から入っている数字は変更できません。

ミス確認のテスト

空きマスにわざと違う数字を入れて、ミス確認 を押してください。間違っている入力済みマスが赤く表示されます。空きマスはミス扱いになりません。

ヒントのテスト

ヒント1、ヒント2、ヒント3 を押すと、現在空いているマスから指定数だけ正しい数字が入ります。

既知の制限

Ver.0.3 は固定問題3つのみです。メモ機能、候補数字、タイマー、問題生成、アカウント、広告、バックエンドは入れていません。ブラウザのツールバーが大きい小型端末では、少しスクロールが残る場合があります。
